- **Logo:** "MASHED" in playful, bold typography with party emoji

## Responsive Design Notes

### Desktop (>1024px)
- Two-column layout for lobby and results
- Larger GIF displays
- Side-by-side submission comparisons

### Tablet (768-1024px)
- Single column, wider elements
- Touch-friendly tap targets (min 48px)

### Mobile (<768px)
- Full-width cards
- Stacked layouts
- Simplified animations
- Bottom sheet for GIF picker
- Floating action buttons

### Accessibility
- All emoji have aria-labels
- Color not the only indicator (icons + text)
- Keyboard navigation support
- Screen reader announcements for phase changes
- Reduced motion option
